## アプリ概要

社内に設置されているコーヒーサーバーを、より一層活用してもらうためのシステム

・コーヒーフレーバーの人気投票機能

・タイムライン機能（コーヒー、お菓子の紹介等）

・管理者用の在庫管理機能

## 起動方法

### `yarn start`

localhost:3000 にて起動

### `yarn mock-api`

localhost:8880 にて起動
