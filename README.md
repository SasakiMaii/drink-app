## アプリ概要

社内に設置されている社内リフレッシュメント施設を、より一層活用してもらうためのシステム

・製品の人気投票機能

・社内交流機能（レコメンド）

・管理者機能

## 起動方法

### `yarn start`

localhost:3000 にて起動

### `yarn mock-api`

localhost:8880 にてモックサーバー起動
